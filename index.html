Create Scenario

Create a new scenario.

  • name Required. Display name of the scenario (String)
  • description Required. For additional information (String)
  • group_name Optional. The name of the scenario's group. Used maps have multiple groups (String)
  • map_id Required. ID of the scenario's map (int)
  • start_id Required. ID of the scenario's start point (int)
  • destination_id Required. ID of the scenario's destination point (int)
  • content Required. The scenario's content (dict):
    • start_section Optional. The map section of the scenario's start point. Used when maps have multiple sections (String)
    • commands Required. List of steps robots need to take to execute the scenario (list):
      • Each step is a tuple of 2 elements
      • 1st element describes the mode e.g. "AMR", "AGV", "ELEVATOR", etc. (String)
      • 2nd element describes the action robots need to do (dict)
      • The mode and action need to be compatible with each other. For example:
        • ("AMR", {"goal_id": "204", "goal_name": "outside elevator"})
        • ("AGV", {"scene": "wait_go_button"})

Modify a specified scenario. All parameters are optional.

  • name Display name of the scenario (String)
  • description For additional information (String)
  • map_id ID of the scenario's map (int)
  • start_id ID of the scenario's start point (int)
  • destination_id ID of the scenario's destination point (int)
  • content The scenario's content (dict):
    • start_section The map section of the scenario's start point. Used when maps have multiple sections (String)
    • commands List of steps robots need to take to execute the scenario (list):
      • Each step is a tuple of 2 elements
      • 1st element describes the mode e.g. "AMR", "AGV", "ELEVATOR", etc. (String)
      • 2nd element describes the action robots need to do (dict)
      • The mode and action need to be compatible with each other. For example:
        • ("AMR", {"goal_id": "204", "goal_name": "outside elevator"})
        • ("AGV", {"scene": "wait_go_button"})
